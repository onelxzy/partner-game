# 今日老婆 Pro (partner-game)

> 基于 [whitefox 的 wife_picker](https://github.com/whitefox20011120/wife_picker) 基础逻辑扩展开发的**修仙 RPG 群聊伴侣大乱斗**互动游戏系统。不仅保留了原版的随机抽取、求婚，更加入了完整的 36 级修仙战力体系、金钱经济系统、好感度护盾机制，让你可以在群里打工赚钱、修仙突破，甚至化身黄毛凭借战力强行抢走别人的老婆！

---

## ✨ 核心特性

- 🎲 **每日随机抽取**：通过 `/今日老婆` 随机抽取一位伴侣，附带 QQ 头像发送。
- 💍 **求婚与结婚系统**：通过 `/娶 @某人` 向指定群友求婚；老婆可以通过 `/甩掉 @老公` 主动解除关系。
- 💰 **经济与打工系统**：每人初始 1000 金币，通过 `/打工` 每日赚取金币（**打工收益将随修仙境界提升而增加！**）。金币用于修仙、陪伴侣约会以及支付抢老婆失败的“治安罚款”。如果你的资产低于 150 金币，连抢老婆的资格都没有！
- ⚔️ **36 级修仙战力系统**：通过 `/修炼` 消耗金币进行境界突破（炼气期->筑基期->...->渡劫圆满）。引入硬核的**“底蕴战力系统”**（战力=等级×(大境界数+1)），境界越高，你在抢老婆或防御黄毛时的战力越具有碾压性优势。突破大境界有概率走火入魔掉级！
- 🛡️ **夫妻同心与好感度护盾**：通过 `/约会` 花费金币增加好感度，每 20 点好感度抵消 1% 的黄毛抢夺成功率。防守时取**夫妻双方中底蕴战力最高者**的数据，老婆也可以成为保护老公的大佬！
- 😈 **霸王硬上弓与黄毛系统**：通过 `/强娶 @某人` 强行绑定单身狗；或通过 `/抢老婆 @某人` 抢走别人的伴侣。受战力压制与好感度影响（无论如何保底10%~90%胜率）。失败将面临高额罚款及长达 1 分钟的自闭冷却惩罚（抢老婆失败还可能受伤掉级！）。**更刺激的是，无论强娶还是抢夺失败，罚款的 50% 都将直接赔偿给受害者/原配老公！** 让你躺着也能发家致富。
- 💸 **存款上交**：通过 `/上交存款 <金额>` 可以把自己辛苦攒下的金币交给伴侣，增加更多互动乐趣与不确定性~
- ⚙️ **高度可配置与持久化**：支持自定义基础抢夺概率、黑名单（绝对强娶失败）、白名单（无限制抽老婆和甩人）等，所有经济和婚姻数据永久保存在 JSON 中。

## 📦 安装

1. 将本插件目录放入 MaiBot 的 `plugins/` 下，建议将文件夹命名为 `partner-game`：

```
plugins/
└── partner-game/
    ├── __init__.py
    ├── _manifest.json
    ├── config.py
    ├── plugin.py
    └── utils.py
```

2. 启动（或重启）MaiBot，插件会被自动加载。
3. 首次启动后会在插件目录下自动创建 `data/partner_game_seen.json` 用于数据持久化。

## ⚙️ 配置说明

您可以通过 MaiBot WebUI 修改，或直接修改系统生成的配置文件。

### 核心参数

| 字段 | 默认值 | 说明 |
|------|--------|------|
| `partner_game.exclude_self` | `true` | 抽取时是否排除机器人自身 |
| `partner_game.exclude_sender` | `true` | 抽取时是否排除指令发起者 |
| `partner_game.admin_users` | `[]` | 管理员 QQ 列表（配置后可使用 `/重置`、`/全服补偿`、`/发钱` 等特权指令） |
| `partner_game.no_limit_users` | `[]` | 白名单 QQ（这些用户抽老婆、离婚、甩人、打劫无冷却和限次，但打工和修炼仍然有限制） |
| `partner_game.cooldown_seconds` | `60` | 普通指令的连续触发冷却时间（秒） |
| `partner_game.force_marry_probability` | `0.4` | `/强娶` 基础成功率（0.0 ~ 1.0） |
| `partner_game.rob_wife_probability` | `0.3` | `/抢老婆` 基础成功率（0.0 ~ 1.0） |
| `partner_game.rob_money_probability` | `0.35` | `/打劫` 基础成功率（0.0 ~ 1.0） |
| `partner_game.forced_wives` | `[]` | 神秘黑幕功能。格式为：`"发起人QQ:老婆QQ"`。涉及到黑幕成员的强娶和抢夺，成功率会强制锁定为 0% 并受到惩罚。 |

## 📁 数据持久化

数据文件：`plugins/partner-game/data/partner_game_seen.json` (视您的文件夹名而定)

包含 `users`（金币和境界）、`marriages`（婚姻关系和好感度）以及 `daily_usage`（限次记录）。每日首次跨日触发会自动清空限次数据，并给保持婚姻状态的玩家增加被动好感度。
