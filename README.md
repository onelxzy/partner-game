# 今日老婆 Pro (partner-game)

> 基于 [whitefox 的 wife_picker](https://github.com/whitefox20011120/wife_picker) 基础逻辑扩展开发**群聊伴侣大乱斗**互动游戏系统。不仅能抽，还能娶、强娶甚至当黄毛抢别人的老婆！附带头像，完整支持伴侣系统互动，内置防重婚、严格技能冷却及惩罚系统。

---

## ✨ 核心特性

- 🎲 **每日随机抽取**：通过 `/今日老婆` 随机抽取一位伴侣，附带 QQ 头像发送。
- 💍 **求婚与结婚系统**：通过 `/娶 @某人` 向指定群友求婚，需对方回复同意；通过 `/我的伴侣` 查看自己当前的对象。
- 😈 **霸道强娶系统**：通过 `/强娶 @某人` 强行绑定对象，有概率失败并进入长达5分钟的自闭冷却惩罚。
- 🤠 **黄毛抢老婆系统**：通过 `/抢老婆 @某人` 抢走别人的老婆！同样具有失败风险和冷却惩罚。
- 💔 **解除关系支持**：老公可以通过 `/离婚` 主动重抽；老婆可以通过 `/甩掉 @老公` 将男方抛弃。
- 🕐 **每日记录与防多角关系**：同一天内重复触发会返回当天已抽到的对象，并且加入了严格的伴侣身份校验防多角关系（防重婚），如果试图对自己的伴侣再次使用技能还会触发专属互动提示。
- ⚙️ **高度可配置**：支持自定义抢夺概率、冷却时间、时区、强制黑幕等。
- 💾 **本地持久化**：抽取记录与离婚记录跨重启保留。

## 📦 安装

1. 将本插件目录放入 MaiBot 的 `plugins/` 下，建议将文件夹命名为 `partner-game` 或 `partner_game_pro`：

```
plugins/
└── partner-game/
    ├── __init__.py
    ├── _manifest.json
    ├── config.py
    ├── plugin.py
    └── utils.py
```

2. 启动（或重启）MaiBot，插件会被自动加载。
3. 首次启动后会在插件目录下自动创建 `data/partner_game_seen.json` 用于数据持久化。

## 🎮 指令玩法

在群聊中可以使用以下指令（私聊会被静默拒绝）：

| 指令 | 别名/简写 | 说明 |
|------|-----------|------|
| `/抽老婆帮助` | `/老婆帮助`, `/今日老婆帮助` | 显示本插件所有功能的帮助菜单。 |
| `/今日老婆` | `今日老婆`, `/抽老婆`, `抽老婆` | 随机抽取一位"今日老婆"；如已有伴侣会被拒绝。 |
| `/娶 @某人` | `娶 @某人` | 向某人求婚，对方需在 60 秒内回复“我同意”或“我拒绝”。 |
| `/强娶 @某人` | `强娶 @某人` | 强行霸占对方为伴侣，概率成功，失败则进入 5 分钟受罚冷却。 |
| `/抢老婆 @某人` | `抢老婆 @某人` | 抢走**对方拥有的伴侣**。概率成功，失败则进入 5 分钟受罚冷却。 |
| `/我的伴侣` | `我的伴侣` | 查看自己当前的伴侣是谁，并标明关系（你是主导方还是被绑定方）。 |
| `/离婚` | `离婚` | 仅限**老公方**（发起方）使用，主动解除关系，每日限一次。 |
| `/甩掉 @老公` | `甩掉 @老公` | 仅限**老婆方**（被绑定方）使用，主动把现任老公甩掉。 |

## ⚙️ 配置说明

您可以通过 MaiBot WebUI 修改，或直接修改系统生成的配置文件。

### 核心参数

| 字段 | 默认值 | 说明 |
|------|--------|------|
| `partner_game.exclude_self` | `true` | 抽取时是否排除机器人自身 |
| `partner_game.exclude_sender` | `true` | 抽取时是否排除指令发起者 |
| `partner_game.no_limit_users` | `[]` | 白名单 QQ（这些用户不受每日一次离婚和抽取限制） |
| `partner_game.cooldown_seconds` | `60` | 普通指令的连续触发冷却时间（秒） |
| `partner_game.proposal_timeout_seconds` | `60` | `/娶` 之后等待对方同意的超时时间（秒） |
| `partner_game.force_marry_probability` | `0.3` | `/强娶` 成功的概率（0.0 ~ 1.0） |
| `partner_game.rob_wife_probability` | `0.2` | `/抢老婆` 成功的概率（0.0 ~ 1.0） |
| `partner_game.forced_wives` | `[]` | 神秘黑幕功能。格式为：`"发起人QQ:老婆QQ"`。 |

## 📁 数据持久化

数据文件：`plugins/partner-game/data/partner_game_seen.json` (视您的文件夹名而定)

每天首次触发任意指令时，会自动清理非今日的过期抽取记录。
